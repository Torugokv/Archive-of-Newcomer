Descrição do curso

Este curso aprsenta conselhos básicos de web e mostra as partes e funções das tags em um documento HTML. Ensina, também, a formatar uma página, inserir imagens e muito mais! Tudo isso, comoexplicações simples e de fácil compreensão.

  2. Módulo 2 - HTML e a web
  3. Módulo 3 - Criando um documento HTML
  4. Módulo 4 - Formatando um documento HTML
  5. Módulo 5 - Criando listas
  6. Módulo 6 - Inserindo Imagens
  7. Módulo 7 - Criando links
  8. Módulo 8 - Combinando tags
  9. Módulo 9 - Usando imagens mapeadas
